# alfabank-limits

Автоматическая установка лимитов на корпоративные карты через Альфа-Бизнес.

## Как работает

1. Запускает локальный HTTP-сервер для приёма SMS-кодов от шлюза
2. Открывает браузер с интерфейсом Альфа-Бизнес
3. Для каждого держателя из конфига находит карту, выставляет лимиты и подтверждает через SMS

## Установка

```bash
pip install -r requirements.txt
playwright install chromium
```

## Конфиг

Скопируй пример и заполни своими данными:

```bash
cp holders_config.example.json holders_config.json
```

Структура `holders_config.json`:

```json
{
  "holders": [
    "ИВАНОВ ИВАН ИВАНОВИЧ"
  ],
  "limits": {
    "all_operations": {
      "day": 100000,
      "month": 500000
    },
    "cash_withdrawal": {
      "day": 50000,
      "month": 200000
    }
  }
}
```

## Запуск

```bash
python main.py
```

### Параметры

| Флаг | По умолчанию | Описание |
|------|-------------|----------|
| `--config` | `holders_config.json` | Путь к конфигу |
| `--port` | `8888` | Порт SMS-сервера |
| `--host` | `0.0.0.0` | Хост SMS-сервера |
| `--debug` | выкл | Подробные логи |
| `--headless` | выкл | Браузер без интерфейса |
| `--sms-token` | нет | Секретный токен для вебхука |

### Пример с параметрами

```bash
python main.py --debug --port 9000 --sms-token my_secret
```

## Настройка SMS-шлюза

После запуска скрипт выведет URL вебхука — укажи его в настройках шлюза как endpoint для POST-запросов.

Поддерживаемые форматы тела запроса:

```json
{ "content": "Код 123456" }
{ "msg": "Ваш код: 123456" }
{ "text": "123456" }
```

Если задан `--sms-token`, шлюз должен передавать его в заголовке `X-SMS-Token`.

## Безопасность

- `holders_config.json` добавлен в `.gitignore` — не коммить реальные данные
- Задавай `--sms-token` чтобы посторонние не могли отправить фиктивный код
- Запускай на машине внутри локальной сети
